Initial release of the unified web search SDK: provider-agnostic `WebSearch` core with the built-in Vercel AI SDK tools adapter (default for `tools()`), optional/env-based API keys (Firecrawl supports keyless), and Firecrawl, Tavily, Exa, Brave, and Serper providers.
